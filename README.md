A simple Flutter application used for tutorial exercises and experimentation.

## Requirements

- Flutter SDK (stable channel)
- Platform tooling for Android, iOS, web, or desktop as needed

## Setup

1. Install Flutter: https://docs.flutter.dev/get-started/install
2. From the project root, fetch dependencies:

```bash
flutter pub get
```

## Run

Run on the connected device or emulator:

```bash
flutter run
```

To run on a specific platform/device, add `-d <device-id>`.

## Project structure

- `lib/`: Dart source, main entrypoint is `lib/main.dart`
- `android/`, `ios/`, `web/`, `macos/`, `windows/`, `linux/`: platform folders
- `test/`: unit and widget tests

## Testing

```bash
flutter test
```
